# 💖 HeartWords AI – Couple Text Rewriter & Shayari Enhancer

Transform simple messages into romantic, flirty, emotional, and creative texts instantly.

**HeartWords AI** is a responsive web application built with **HTML, CSS, and JavaScript** that helps couples rewrite boring messages and improve their shayari with different writing styles.

Whether you want to send a heartfelt message, create a cute caption, or enhance your shayari, HeartWords AI makes it simple and fun.

---

## ✨ Features

* ❤️ Rewrite simple text into romantic messages
* 😉 Generate flirty and playful responses
* 🌹 Improve shayari and poetic content
* 🥰 Multiple writing styles and tones
* 🌙 Dark and light mode support
* 📱 Fully responsive design
* 📋 One-click copy functionality
* ⭐ Save favorite messages locally
* 🔄 Regenerate multiple variations
* 🎨 Modern glassmorphism user interface
* ⚡ Fast performance with Vanilla JavaScript
* 💾 Theme and favorites saved using Local Storage
